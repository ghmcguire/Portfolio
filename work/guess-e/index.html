Guess-E

Guess-E is like Pictionary powered by AI.

Connect you phone with some friends' and start a round. One person comes up with a prompt for an image to generate and the Dall-E 2 API will generate an image based on the prompt. The other players will then try to guess what the prompt was.

Guess-E was built with Swift and SwiftUI in Xcode. It uses the Dall-E 2 API from OpenAI to generate images.

My role was as the head designer and a front end developer. I designed the layout and general styling in Figma. Other developers implemented most of the layouts and I added the styling and animations.
