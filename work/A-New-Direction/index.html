A New Direction

A New Direction is a clone of BitLife powered by ChatGPT.

It is a text based game where you can live out your life in a variety of ways.

My focus was on backend. I used Node.js and Express to create a wrapper API to interface with out frontend and the GPT API. I also helped build the frontend using HTML, CSS, and JS. I wrote a lot of the text that the AI uses to generate the game.
